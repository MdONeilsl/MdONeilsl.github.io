How to use:

            
    
                
  • Use #define word __UID__ to define any
                    word as a unique short ID
    • 
                
  • Use #define word replacement for
                    custom replacements
    • 
                
  • Class name replacements only affect class="..." in HTML nodes
    • 
                
  • Unused CSS rules are automatically removed
    • 
                
  • UIDs contain letters and digits, but always start with a letter
    • 
            

            
// Definition
#define coolclass __UID__
#define --main-clr __UID__

// CSS Rules
.coolclass { color: var(--main-clr); }

// HTML (Optional)
<div class="coolclass" id="coolclass">Test</div>
