Quick Guide

  • Upload image for canvas texture
  • Draw 2D shapes, apply textures, and export as 3D mesh for Blender
  • Select a shape tool and click+drag to draw
  • Click shapes to select (highlighted in red)
  • Drag selected shapes to reposition
  • Hold S + drag to scale
  • Hold R + drag to rotate
  • Ctrl+C to copy, Ctrl+V to paste
  • Delete to remove selected shapes
  • Export mesh - save texture with .dae file
